How it works:
1. Add one or more cards + pick their edition/set from the dropdown (or type a custom version).
2. Click Search TCGPlayer — this calls your local proxy server.
3. Or run python scraper.py --cards "Black Lotus:Alpha" "Tropical Island:Unlimited" --game mtg --output combo.json and load the file here.
The Bundles tab shows sellers who carry all cards, ranked cheapest first.
